# 索引目录

本目录提供两种检索入口，帮助你在 Skill-hub 上快速找到所需技能：

| 索引 | 用途 | 适合场景 |
|---|---|---|
| [by-name.md](./by-name.md) | 按技能名 A-Z 全量索引 | 知道技能名时直接定位 |
| [by-domain.md](./by-domain.md) | 按 7 个能力域分组 | 按“做什么”查找，如 Agent 工程、行业业务 |
| [skills-lock.json](./skills-lock.json) | 机器可读映射表 | 脚本同步、自动化使用 |
| [external-workflow-packs.json](./external-workflow-packs.json) | 外部完整工作流包目录 | 发现、安装和维护不应拆分的第三方工作流包 |

当前共 70 个技能（62 个共享 + 8 个项目私有）。

## 命名与分类约定

见 [../docs/directory-conventions.md](../docs/directory-conventions.md)。

完整的第三方工作流包不属于单 skill 索引；见 [../docs/external-workflow-packs.md](../docs/external-workflow-packs.md)。
